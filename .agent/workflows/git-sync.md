---
description: Sync local changes with remote repository (Pull & Push)
---

1. Pull latest changes from remote

   ```bash
   git pull
   ```

2. Stage all local changes

   ```bash
   git add .
   ```

3. Commit changes (Update message as needed)

   ```bash
   git commit -m "<What is the changes from last commit?>"
   ```

4. Push changes to remote
   ```bash
   git push
   ```
